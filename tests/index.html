Implementation in Python

This project implements a queue using Python. The queue supports the following operations:

  • Enqueue: Add an element to the end of the queue.
  • Dequeue: Remove the element at the front of the queue.
  • Peek: Get the value of the front element without removing it.
  • IsEmpty: Check if the queue is empty.

Example Usage

                queue = Queue(capacity=2)
                queue.enqueue(1)
                queue.enqueue(2)
                print(queue.peek())  # Output: 1
                queue.dequeue()
                print(queue.is_empty())  # Output: False
                print(queue.is_Full())  # Output: True
